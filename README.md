# Retail-Sales-Analysis
Exploratory data analysis of a retail sales dataset using Python, Jupyter notebook (Pandas, Matplotlib, Seaborn) and Power BI — covering customer demographics, product trends, and seasonal patterns.

# Retail Sales Analysis Dashboard

## Project Overview
This project analyzes retail sales data using Python and Power BI to uncover insights into customer behavior, sales performance, and product category trends.

## Tools Used
- Python
- NumPy
- Pandas
- Matplotlib
- Jupyter Notebook
- Power BI

## Key Analysis
- Monthly Sales Trends
- Product Category Performance
- Gender-wise Spending Analysis
- Customer Count Analysis
- Age vs Spending Patterns

## Dashboard Features
- Interactive Power BI Dashboard
- Sales Trend Visualization
- Customer Demographic Insights
- Category-wise Revenue Analysis

## Key Insights
- Identified peak sales periods.
- Compared spending patterns across genders.
- Analyzed category-wise revenue contribution.
- Explored age-related purchasing behavior.
